# Unreal Engine Editor

## Overview

The Unreal Engine Editor is the primary environment for creating, editing, and managing game projects. It provides a collection of tools that allow developers to build levels, configure assets, and design gameplay.

---

## Main Editor Panels

### Details Panel

The **Details Panel** displays and edits the properties of the selected Actor.

**Common Uses**

- Transform
- Materials
- Physics
- Collision
- Rendering

---

### World Outliner

The **World Outliner** lists every Actor in the current level.

It helps developers:

- Locate Actors
- Organize hierarchies
- Search objects
- Manage folders

---

### Content Browser

The **Content Browser** stores and organizes project assets.

Examples:

- Static Meshes
- Textures
- Materials
- Blueprints
- Audio
- Animations

---

### Output Log

The **Output Log** displays:

- Messages
- Warnings
- Errors

It is commonly used for debugging projects.

---

### Material Editor

The **Material Editor** is used to create and edit materials.

Developers can create node-based materials that define the appearance of surfaces.

---

### Sequencer

**Sequencer** is Unreal Engine's cinematic editor.

It allows developers to create:

- Camera animations
- Character animations
- Cutscenes
- Timeline-based events

---

## Summary

| Panel | Purpose |
|--------|---------|
| Details Panel | Edit Actor properties |
| World Outliner | Display Actors hierarchically |
| Content Browser | Manage assets |
| Output Log | Display debugging information |
| Material Editor | Create materials |
| Sequencer | Create cinematics |

---

## Key Takeaways

- The Editor is the main workspace of Unreal Engine.
- Each panel has a specific role.
- Understanding the editor improves workflow efficiency.
