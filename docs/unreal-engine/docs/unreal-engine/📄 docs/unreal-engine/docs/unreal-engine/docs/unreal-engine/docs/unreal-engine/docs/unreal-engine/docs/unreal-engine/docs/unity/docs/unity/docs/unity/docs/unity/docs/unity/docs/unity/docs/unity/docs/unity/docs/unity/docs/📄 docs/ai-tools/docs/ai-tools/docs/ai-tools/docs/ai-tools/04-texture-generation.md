# What are PBR Textures?

Physically Based Rendering (PBR) uses multiple texture maps to simulate realistic surfaces.

Common texture maps include:

- Base Color (Albedo)
- Normal Map
- Roughness
- Metallic
- Ambient Occlusion (AO)
- Height / Displacement
