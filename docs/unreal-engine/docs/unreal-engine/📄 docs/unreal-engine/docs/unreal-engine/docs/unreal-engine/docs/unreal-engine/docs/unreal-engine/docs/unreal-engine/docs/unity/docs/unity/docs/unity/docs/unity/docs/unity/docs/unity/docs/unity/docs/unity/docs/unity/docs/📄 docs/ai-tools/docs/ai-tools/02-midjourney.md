# Overview

Midjourney is an AI-powered image generation tool that creates high-quality artwork from text prompts.

In game development, it is commonly used for concept art, character design, environment exploration, mood boards, and visual prototyping.

# What is Prompt Engineering?

Prompt Engineering is the process of writing clear and descriptive instructions to guide AI image generation.

A good prompt usually includes:

- Subject
- Environment
- Art Style
- Lighting
- Camera Angle
- Quality
- Aspect Ratio

# Best Practices

✔ Write clear prompts.

✔ Specify the art style.

✔ Include lighting information.

✔ Define camera angles.

✔ Iterate multiple times.

✔ Save successful prompts.

# Common Mistakes

❌ Writing vague prompts.

❌ Forgetting aspect ratios.

❌ Mixing incompatible art styles.

❌ Expecting perfect results on the first attempt.
