# Overview

MetaHuman is Epic Games' digital human creation framework. It enables developers to create highly realistic human characters for games, cinematics, virtual production, and interactive experiences.

MetaHumans are fully rigged, animation-ready, and seamlessly integrated with Unreal Engine.

# What is MetaHuman?

MetaHuman allows developers to build realistic digital humans without creating every detail from scratch.

Characters include:

- High-quality facial features
- Detailed skin shaders
- Hair and eyebrows
- Eyes and teeth
- Fully rigged skeleton
- Facial animation support

# Main Features

## Realistic Characters

MetaHuman provides high-quality human models suitable for modern games and cinematics.

---

## Facial Rig

Every MetaHuman includes a facial rig for detailed expressions.

Examples:

- Smiling
- Talking
- Blinking
- Emotional expressions

---

## Body Rig

Characters include a complete body skeleton compatible with Unreal Engine animation systems.

Supports:

- Walking
- Running
- Jumping
- Combat
- Motion Matching

---

## Hair System

MetaHuman uses Unreal Engine's Groom System.

Supports:

- Hair
- Beard
- Eyebrows
- Eyelashes

---

## Clothing

Characters can be customized with different clothing assets and accessories.
